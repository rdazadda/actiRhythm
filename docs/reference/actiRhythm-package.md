# actiRhythm: Circadian Rest-Activity Rhythm Analysis

Quantifies the circadian rest-activity rhythm of a single subject from
activity counts: nonparametric measures, cosinor models with a
rhythmicity test, period estimation, fractal and nonlinear measures, the
Sleep Regularity Index, and phase metrics. Analyses run on an
activity-count vector and its timestamps, and a built-in reader loads
'ActiGraph' '.agd' files directly.

# Complete Circadian Analysis

Complete Circadian Analysis

## Usage

``` r
circadian_cpp(minute_data, hours_per_day = 24L, start_minute = 0L)
```

## Arguments

- minute_data:

  Numeric vector of minute-level activity

- hours_per_day:

  Hours per day (default: 24)

- start_minute:

  Start minute of day (0-1439)

## Value

List with L5, M10, RA, IS, IV, phi

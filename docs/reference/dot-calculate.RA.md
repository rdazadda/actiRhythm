# Calculate Relative Amplitude

RA = (M10 - L5) / (M10 + L5)

## Usage

``` r
.calculate.RA(M10, L5)
```

## Arguments

- M10:

  Most active 10-hour average

- L5:

  Least active 5-hour average

## Value

Numeric. Relative amplitude (0-1)

## References

van Someren EJ, et al. (1999). Chronobiol Int, 16(4):505-518.

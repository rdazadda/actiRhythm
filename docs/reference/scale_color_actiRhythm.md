# ggplot2 Scale for actiRhythm Colors

Discrete color scale using the actiRhythm palette.

## Usage

``` r
scale_color_actiRhythm(type = "categorical", ...)
```

## Arguments

- type:

  Character. Palette type (see actiRhythm_colors)

- ...:

  Additional arguments passed to ggplot2::scale_color_manual

## Value

A ggplot2 scale object

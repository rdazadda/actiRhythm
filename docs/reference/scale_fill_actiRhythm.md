# ggplot2 Fill Scale for actiRhythm Colors

Discrete fill scale using the actiRhythm palette.

## Usage

``` r
scale_fill_actiRhythm(type = "categorical", ...)
```

## Arguments

- type:

  Character. Palette type (see actiRhythm_colors)

- ...:

  Additional arguments passed to ggplot2::scale_fill_manual

## Value

A ggplot2 scale object
